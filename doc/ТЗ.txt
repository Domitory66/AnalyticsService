Разработать сервис аналитики.
Данный сервис должен принимать на вход данные от SHWService о подключении нового устройства с указанием ip адреса, порта и имени файла для обработки
 

Общение с SHWService:
1. Добавить камеру
2. Удалить камеру
3. Отправить видеопоток

 1.Подключиться к камере. 
 2. Считать поток видеоизображения.
 3. Обработка видеопотока
    3.1 Анализ видео потока на наличие жеста руки.
    3.2 Добавление квадрата и наименования жеста на видео при обнаружении 
 4. Передача изменённого видеопотока на отображение с указанием ip камеры, порта.
